# Martin Fowler

## Quem é

Autor, palestrante e chief scientist da Thoughtworks. Referência mundial em arquitetura de software, padrões de design e práticas de engenharia.

## Contribuição relevante para esta wiki

Co-autor (com Birgitta Böckeler) do artigo "Harness Engineering for Coding Agent Users" (2026), que define:
- A fórmula Agent = Model + Harness
- Distinção entre **guides** (feedforward) e **sensors** (feedback)
- Conceito de "positive prompt injection" (sensor que ensina o agente)
- Architecture fitness harness
