# GitHub Copilot

## O que é

Ferramenta de autocomplete e review de código da GitHub/Microsoft. Autocomplete-first, menos agentic que Claude Code.

## Características

- Integrado ao ecossistema GitHub
- Autocomplete-first
- Menos agentic, mais discreto
- Copilot Code Review: saiu do zero em abril/2025, ultrapassou CodeRabbit em alcance em 7 meses
